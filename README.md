# Password Generator

A simple Python tool that asks the user for a password length and generates a random password using letters and numbers.

## Key Skills

* Importing Modules (random, string)
* String Manipulation
* User Input Handling
* Loops
* Basic Python Programming

## Why This Project?

This project helps beginners understand how Python's built-in libraries can be used to solve real-world security problems. It introduces the concept of generating random and unpredictable passwords, an important aspect of cybersecurity.

## Features

* User chooses the password length.
* Generates a random password.
* Uses uppercase letters, lowercase letters, and numbers.
* Beginner-friendly and easy to understand.

## Technologies Used

* Python 3
* random module
* string module

## Sample Output

```
Enter password length: 8
Generated Password: A7kP2mX9
```

## Learning Outcomes

By building this project, you will learn:

* How to import and use Python modules.
* How to manipulate strings.
* How to generate random values.
* How to create a simple real-world utility tool.
